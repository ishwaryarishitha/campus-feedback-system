# Campus Feedback Management System

A Django-based web application designed to collect and manage student feedback
for teachers and courses in an academic environment.

## Features
- Student feedback submission
- Teacher-wise feedback collection
- Admin dashboard for managing feedback
- Secure authentication
- Clean UI using Django templates

## Tech Stack
- Python
- Django
- SQLite
- HTML, CSS, Bootstrap

## Setup Instructions
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies using `pip install -r requirements.txt`
4. Run migrations using `python manage.py migrate`
5. Start the server using `python manage.py runserver`
